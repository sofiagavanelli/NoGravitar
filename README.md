# No-Gravitar

## Installazione

### Linux
```
sudo apt-get install g++
sudo apt-get install libsfml-dev
git clone https://github.com/sofiagavanelli/No-Gravitar.git
# all'interno della directory del progetto
cd code
make
./game
```

### Windows
```
git clone https://github.com/sofiagavanelli/No-Gravitar.git
# all'interno della directory del progetto 
cd windows
# open file NonGravitar.exe
```
